什么时候使用

两者都是用来描述概念的工具，如形状Shape、食物Food等，现实中找不到对应的物体，只存在于我们脑中，这也是为什么他们无法用来创建对象。

区别

抽象类是描述整体，接口是描述部分。所以，一个类只能继承一个抽象类，但是却能实现多个接口。

接口的成员只能是public，接口的字段只能是final与static。为什么？
